# DnD 5 2014 BagarreSandwich


# Fonctionnalités prévues
Bon l'objectif c'est d'avoir sous la main pendant les sessions une interface graphique rapide à utiliser qui me permette d'estimer les longueurs sur la carte :
Il faut que je puisse choisir le mouvement de chaque perso/créature puis le déplacer tout en tenant compte des terrains difficiles et autres obstacles (qu'il faut donc rajouter)
Aussi ce serait pas mal de rajouter les sorts à effet de zone pque je peux placer dans la portée possible. Et si jamais ils déplacent des creatures ou modifient le terrain, il faut le prendre en compte
La taille des perso et leur allonge, ainsi que l'impossibilité de se traverser est aussi à ajouter.
Un autre truc à implémenter est le retour en arrière : si je fais une mauvaise manip que je puisse undo.

Une interface aussi schématique que possible au final, le but n'est pas de remplacer le combat sur papier mais de simplifier les estimations à la louche de qui tu touche ou jusqu'où tu va.
